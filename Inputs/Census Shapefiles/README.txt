Census geography files to give the data a spatial component. This will be used to 
aggregate the data to a census tract level.

The shapefiles are downloaded from here: https://www.census.gov/cgi-bin/geo/shapefiles/index.php 

The blocks shapefiles was to big to share via a zipped folder through github, so it will 
need to be downloaded and placed into a folder named Blocks within each of the state folders.
